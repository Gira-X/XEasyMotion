# XEasyMotion
using keyboard control mouse on MacOS ( OS X ), like keynav.

## Download
https://github.com/h2ero/XEasyMotion/releases

## Usage

* `hjkl` zoom area
* `shift+hjkl` move area
* `enter` click
* `shift enter` right click
* `cmd+i` call XEasyMotion
